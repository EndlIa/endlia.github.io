这是第一篇测试用博客。它的目的很简单：确认 Hugo 项目、黑白极简主题、文章列表和文章详情页都可以正常工作。

## 为什么用 Hugo

Hugo 的优势是快、稳定、部署简单。文章用 Markdown 编写，推送到 GitHub 后由 GitHub Actions 自动构建，再发布到 GitHub Pages。

## 后续可以写什么

- 技术笔记
- 项目复盘
- 读书摘录
- 日常想法

先把写作流程跑通，比一开始追求复杂功能更重要。
